💻 Run locally on your PC
After downloading and extracting the ZIP, open a terminal in the bot folder and run:
Step 1 — Install dependencies
npm install
Step 2 — Start the bot
node index.js
Windows: Right-click inside the bot folder → Open in Terminal
Linux/Mac: cd ~/Downloads/bot-folder then run the commands above
